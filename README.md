## Overview
A Java-based college database application that models students, faculty, courses, sections, and enrollments using object-oriented design.

## Features
- Object-oriented modeling of core academic entities  
- Structured relationships between students, courses, and faculty  
- Centralized database-style management logic  
- Console-based execution for testing and interaction  

## Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- Data Modeling
